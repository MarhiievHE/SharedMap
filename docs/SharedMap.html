Class: SharedMap

SharedMap(maxSize, keySize, objSize) → {SharedMap}

SharedMap zero-dependency Vanilla JS implementation of SharedMap, a synchronous multi-threading capable, fine-grain-locked with deadlock recovery, static memory allocated, coalesced-chaining HashMap, backed by SharedArrayBuffer that supports deleting and is capable of auto-defragmenting itself on delete unless almost full compatible with both Node.js and SharedArrayBuffer-enabled browsers
